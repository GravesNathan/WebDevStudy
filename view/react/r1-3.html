Section 1: React Introduction

Local Environment Setup

Now get get to start the real fun! There are a few methods to setting up a local environment to learn React. What's important to understand is that React is a server side framework. It's also important to know that, as with other frameworks, React is actually not understood by any browsers.

To successfully implement React you're going to need to have a local install which will act as your server. One way of doing this is to find, download, and install, each of the required frameworks used by React. Depending on the version installed the plugins required can vary as well.

To simplify our course we'll be installing React by utilizing a handy pre-built package as detailed below.

  1. Download and Install NodeJS
    The package for our next step requires npm. Since NodeJs includes npm I've included this as our first step. If you already have npm working in your environment you can skip step 1.

    The version I'll be using for this tutorial is 10.15.3 LTS. It is generally recommend you stick with the current recommended version, but if you have trouble and need an old version you can check here.

  2. Prepare a "creator" command
    Open up the shell for your computer, in Windows this would be Powershell or command prompt. Issue this command (or the appropriate command for your OS for creating global variables).

    npm install -g create-react-app

    This will create a global variable/command that can be used to create a skeleton react application with ease.

  3. Install React Developer Tools (highly recommended)
    As was mentioned previously the browser doesn't understand ReactJS. This can make it difficult to troubleshoot when a page isn't working as expecting. To assist us in this course I recommend installing React Developer Tools.

    The tool can be found on Github. If you're using Google Chrome then you can simply install the extension here.

  4. Create a React App
    Navigate to the directory just above where you want your React App to be located. From there type the following command.

    create-react-app myapp

    This command can take several minutes to execute. It goes through all the downloads to setup a skeleton React application. "myapp" will become the root directory of your app and will exist in a folder directly below the directory the command is executed in.
