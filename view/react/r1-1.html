Section 1: React Introduction

What is React?

React or ReactJS is a web development library based on JavaScript. The purpose of React is to simplify the building of dynamic webpages; with a focus on front-end development. Engineers at Facebook started development on more efficient ways to maintain their website around 2011 (depending on what you consider part of React). This eventually led to the development, and thankfully open-sourcing, of React in 2013.

Despite the relatively short number of years since it's development, React is quckly becomming a popular tool among front-end web developers. Although not required for web development, if your goal is to become a web developer (especially on the front-end) this and similar courses come highly recommended.
