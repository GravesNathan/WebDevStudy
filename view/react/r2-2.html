JSX Elements

JSX Elements are the most basic piece of JSX. Generally they look exactly, or almost exactly, like their HTML element counterparts. Check out this example <h1> JSX element example.

<h1>Hello world</h1>

See any difference between this h1 JSX element and an h1 element in HTML? No...? Well good, because you shouldn't. In fact the only noticible difference between JSX elements and their corresponding HTML elements is JSX Elements are found in JavaScript files!

You should also know that JSX Elements can be used anywhere that traditional JavaScript can be used. For example, we can save JSX Elements to variables or constants in JavaScript.

var navBar = <nav>I am a nav bar</nav>;
