Let's go over the first few lines in detail.

import and from

"import" is an ES6 function that allows us to bring code from other modules into our current scope. It is often used in conjunction with the "from" command which indicates where to look for the module. This works essentially the same with Reactjs, except that we can also import other Reactjs component which we'll get into later.

import React and ReactDOM

All you really need to know about these lines is we are importing a module (or component) named "React" from a location of "react". The same is true of ReactDOM from react-dom. import React is necessary for the Reactjs to be understood by the application. import ReactDOM is necessary to view (or render) React content on our page. Don't worry too much about understanding the details at this point.
